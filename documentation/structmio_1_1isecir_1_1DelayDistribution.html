Detailed Description

Function describing the time spent in a compartment before transiting to next compartment.

This class defines a function that specifies which proportion of individuals is still in the compartment after a certain state_age (i.e. time after entering the compartment) and has not yet progressed to the next. See also function \gamma in Overleaf. Currently, you can only use a smoother_cosine() function with different parameters for this purpose. TransitionDistribution implements a vector with DelayDistributions for each compartment.
