Detailed Description

template<class PtrIter>
class epi::PointerDereferencingIterator< PtrIter >

iterator adaptor that makes a range of T* look like a range of T e.g.

std::vector<int*> v; then *PointerDereferencingIterator<...>(v.begin()) == **v.begin()
