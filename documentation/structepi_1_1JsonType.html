template<class T, class = void>
struct epi::JsonType< T, class >

JsonType allows the conversion of basic types for serialization.

All types that are directly handled by the json serializer inherit from std::true_type. They also define the transform functions for conversion from and to Json::Value. All others inherit from std::false_type, external serialization functions need to be available.
